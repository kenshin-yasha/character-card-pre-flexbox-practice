# Learning CSS before moving on the Flexbox, Grid etc.

## What I Learned

### **HTML Skills**
- Creating structured content using `div`, `h2`, `h3`, `p`, `img`
- Grouping elements logically (card → image → buttons → details)
- Writing clean, readable HTML

### **CSS Layout Skills**
- Centering content using `margin: auto`
- Limiting width using `max-width`
- Using `box-sizing: border-box` for predictable layouts
- Applying padding, margin, and spacing consistently
- Creating a scrollable container using `max-height` + `overflow-y: auto`
- Styling buttons with transitions and hover effects
- Making circular profile images with `border-radius: 50%
